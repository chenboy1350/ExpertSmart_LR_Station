LR-Station ระบบจัดการห้องคลอด
Project นี้จะแบ่งออกเป็น 2 ส่วนคือ หน้าบ้านและหลังบ้าน
โฟลเดอร์ LR-Station เป็นโปรเจคที่ทำหน้าที่เป็นหน้าบ้าน พัฒนาด้วย ASP.NET เป็นเว็บที่ใช้จัดการข้อมูลทั่วไป
โฟลเดอร์ LR-WebAPI เป็นโปรเจคที่เป็นหลังบ้าน เป็น API ที่ใช้ในการ Query Database ด้วยข้อมูลที่ส่งมาจากหน้าบ้านอีกทีหนึ่ง พัฒนาด้วย .NET Core
